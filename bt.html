蓝牙音响 不再断开

Windows 后台应用，每隔指定秒数播放静音音频，保持蓝牙音响连接不断开。
彻底告别"暂停一会儿就断连"的烦恼。

免费下载 GitHub 源码
🔊

痛点问题

蓝牙音响在暂停播放后，系统会在几十秒后自动断开连接。每次听歌、看视频中途停顿一下，音响就断了，需要重新手动连接，非常麻烦。

✨

解决方案

程序在系统托盘后台运行，每隔 30 秒播放一次极短静音音频，让蓝牙音响保持"活跃"状态，系统就不会自动断开连接。全程无声，不会打扰你。

功能特点

💻

系统托盘运行

最小化到托盘，不占任务栏，后台静默运行，完全无感。

🔄

自动保持连接

每 30 秒自动触发一次静音音频，蓝牙音响持续活跃。

⚡

开机自启

右键菜单一键设置开机自动运行，一劳永逸。

⚙️

自定义间隔

自由调节播放间隔（5~120秒），适应不同蓝牙设备。

🖥️

原生对话框

使用 Windows 原生 API，稳定可靠，无事件循环冲突。

📦

独立 exe

无需安装 Python 环境，下载解压后双击即用。

工作原理

⏰

定时触发

启动计时器，每 N 秒触发一次保活信号

🔇

静音播放

播放极短静音 WAV 音频（约 0.15 秒）

🧐

设备激活

蓝牙音响被激活，保持连接活跃状态

✅

防止断连

系统判定设备"活跃中"，不会自动断开
